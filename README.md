This repository now keeps **client usage documentation only**.

- Client runtime usage

- Activation and sync notes for end users

Chinese note:

- 该仓库当前仅保留客户端使用说明，不再提供服务端/发布流程文档。

## Quick Start

```powershell
python -m venv .venv
.\.venv\Scripts\activate
python -m pip install -U pip
python -m pip install PyQt6==6.8.1 PyQt6-Qt6==6.8.2 PyQt6-sip==13.10.0
python main.py
```
